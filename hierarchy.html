Tentang Memory Hierarchy

Hierarki memori adalah organisasi berbagai jenis memori berdasarkan kecepatan, kapasitas, dan harga.

Level 1: Register

Paling cepat (<1 ns), paling kecil (bytes), paling mahal. Di dalam CPU.

Level 2: Cache

Sangat cepat (1-10 ns), kecil (KB-MB), mahal. L1, L2, L3 cache.

Level 3: Main Memory (RAM)

Cepat (~100 ns), sedang (GB), harga moderat. Primary storage.

Level 4: Secondary Storage

Lambat (~10 ms), besar (TB), murah. HDD/SSD persistent storage.
